## /shot list
Lists available weapons, excluding the ones that are hidden.
### Syntax
```/shot list <all/page>
```
### Parameters
- **all** - Lists all weapons.
- **page** - Lists weapons by page number.
!!! example
    ```/shot list all
    ```
